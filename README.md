## 📋 Tentang Project

**Orbit Kicks** adalah platform e-commerce sneakers premium yang dirancang untuk memberikan pengalaman berbelanja digital yang imersif dan berkesan.

## 🎯 Fitur Utama

### 🎨 UI/UX (Neobrutalism)
- Desain bold dengan border hitam tebal & warna solid
- Smooth page transitions (Framer Motion)
- Stagger animations & responsive layout

### 👟 Product Experience
- Tilt 3D effect + zoom on click
- Size selector pill buttons
- Add to cart dengan particle burst animation

### 🛍️ Shopping Features
- Product grid & detail page dengan animated sidebar
- Cart system
- Midtrans payment integration (mode test)

### 🎬 Interaksi
- Hover scale + border tebal pada cards
- Hero section dengan typography bold & CTA kontras
- Image hover zoom

## 🛠️ Tech Stack

| Kategori | Teknologi |
|----------|-----------|
| **Framework** | Next.js 16 (App Router) |
| **Language** | TypeScript (strict mode) |
| **Styling** | Tailwind CSS |
| **Animations** | Framer Motion, react-parallax-tilt |
| **UI Components** | Custom + shadcn/ui |
| **Icons** | Lucide React |
| **Package Manager** | npm |
| **Deployment** | Vercel |

## 🚀 Cara Menjalankan

```bash
# Clone repository
git clone https://github.com/ViannKia/orbit-kicks.git

# Masuk ke folder project
cd orbit-kicks

# Install dependensi
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build
npm start

```
