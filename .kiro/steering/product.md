# Orbit Kicks - Product Vision

## Core Value Proposition
Premium sneakers e-commerce dengan pengalaman 3D immersive. Setiap sneakers bisa dilihat 360° dengan rotasi smooth dan lighting cinematic.

## Target Audience
- Sneaker enthusiasts (18-35 tahun)
- Collectors yang menghargai detail
- Gen Z / gamers yang terbiasa dengan visual 3D

## MVP Features (Priority Order)
1. 360° product viewer dengan drag-to-rotate
2. Smooth thumbnail navigation (ganti model 3D saat klik warna/variant)
3. Cinematic entrance animation untuk product hero
4. Micro-interactions pada hover (zoom glow, scale effect)
5. Add to cart dengan particle effect

## Product Categories (Fase 1)
- Nike (Air Jordan, Dunk, Air Max)
- Adidas (Yeezy, Samba)
- New Balance (990 series)

## Visual Style — Neobrutalism
- **Theme**: Neobrutalism (cream background, hard black borders, offset drop shadows, no blur)
- **Background**: Cream (`#fef6e4`) main, electric yellow (`#fff200`) accent sections
- **Borders**: 3-4px solid black on all interactive elements
- **Shadows**: Hard offset (`8px 8px 0 #000`), no blur
- **Accent palette**: Hot pink `#ff5c8d`, electric yellow `#fff200`, cyan `#00d9ff`, lime `#a3ff00`, orange `#ff7a00`
- **Typography**:
  - Display: Archivo Black (uppercase, ultra-bold for headlines)
  - Body: Space Grotesk (clean geometric sans)
- **Card style**: Solid color fills (not glassmorphism), thick borders, hard offset shadows, slight stickering rotations
- **Hover pattern**: button/card shifts to shadow position (translate by offset, shadow shrinks to 0) — a tactile "press"

## Non-Goals (Not for MVP)
- No real checkout/payment (mock cart dulu)
- No user auth
- No admin dashboard
